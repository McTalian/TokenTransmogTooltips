> **Incremental Workflow**: Fill sections progressively, then use `/tttgen` output to populate Appearance/ModID data. Focus your time on gathering IDs from in-game sources.

**Recommended Steps**:
1. Fill "Raid Information", "Token Groups & Class Mappings", "Difficulties", and "Token Names & Slots"
2. Run `$stub [ID]` to generate Token Item ID Mapping stubs
3. Fill "Token Item ID Mapping" from the Dungeon Journal
4. Use `/tttgen` (alpha builds) to generate raw output
5. Paste the full `/tttgen` output into the RAW OUTPUT section and mark selections with `[X]`
6. Run `$plan [ID]` ‚Üí generates the PLAN OUTPUT section
7. Review PLAN OUTPUT and run `$generate [ID]` ‚Üí generates code
